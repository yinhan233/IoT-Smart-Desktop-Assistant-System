# 智教多维感知系统 | SmartTeach Multidimensional Perception System
课堂多模态行为智能分析与教学辅助平台

## 项目简介
本项目以“多维感知”为核心，通过**STM32+ESP32+Web端**分工协作，实现对学生课堂状态的全维度分析与教学辅助：
- **视觉感知**：低头/抬头率实时统计、专注/分心表情识别
- **听觉感知**：课堂互动音量监测、发言频率统计
- **智能建议**：基于多模态数据自动生成课堂调整策略

## 功能特点
- 📊 多维度数据实时可视化
- 🎯 轻量级算法落地
- 💻 教师端简洁Web界面，支持一键查看课堂报告
- 🔧 双芯片分工：STM32负责底层控制，ESP32负责图像/音频采集与Wi-Fi传输，硬件成本低
